# siggraph-papers-schedule

Visit <https://keenancrane.github.io/siggraph-papers-schedule/>

A more usable version of the SIGGRAPH Technical Papers schedule, as well as scripts to extract/generate pages like this from future schedule webpages.  Displays papers in Fast Forward order, and makes it easy to select and share schedules, including automatic conflict identification.  In future years, can be re-built from official ACM pages (Technical Papers Program listing and/or Fast Forward Session page).
